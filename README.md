# Student Manager System 🎓

A simple JavaScript project that manages student data using an **Array of Objects**.
The program demonstrates how to store structured data, process it with loops, and perform operations such as calculating averages, searching, and updating data.

This project focuses on strengthening logical thinking and understanding how real programs manage collections of data.

---

## 📌 Features

* Display all students with their grades
* Calculate the **average grade**
* Find the **highest grade**
* Find the **lowest grade**
* Show **students who passed**
* Search for a student by name
* Add a new student to the list
* Remove the last student from the list

---

## 🧠 Concepts Used

This project practices important JavaScript fundamentals:

* Arrays
* Objects
* Array of Objects
* Loops (`for`)
* Conditional Statements (`if`)
* Variables
* Array Methods (`push`, `pop`)
* Basic Data Processing
